# Manga grammar

- Give each beat one dominant narrative purpose.
- Alternate establishing, medium, and detail emphasis; do not make every beat a
  close-up or every page a grid.
- Use page turns for reveals and earned cliffhangers, not arbitrary truncation.
- Reserve dense explanation for calm visual beats and split overloaded beats.
- Preserve reading order, speaker clarity, and spatial continuity.
- Visual intent describes composition and emotion, never executable layout code.
- Repeated character cues must match accepted canon and reusable asset metadata.
